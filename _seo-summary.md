## 6. 可抄作业总清单

### 页面结构（抄 Andersen）

```
首页 → 产品系列 → 产品详情 → FAQ → 博客 → 案例 → 关于 → 联系
       ↓           ↓          ↓       ↓       ↓
    3-4 级阶梯   1000+ 字   30+ Q&A  每周1篇  挑战→方案→效果
```

### Title 模板（抄 Andersen，改亿合/万加的错）

```
产品页:  [系列] [产品类型] | [核心特性] | [品牌]
博客:    [标题 — 含主关键词] | [品牌] Blog
FAQ:     [完整自然问句] | [品牌] Support
首页:    [品牌] — [一句话价值主张] | [核心关键词]
```

### FAQ 架构（综合 Andersen + 我们已有的 35 条 FAQ）

```
8 大分类 × 3-5 条 × Schema = 搜索引擎可见的知识资产
```

## 7. 核心结论

1. **Andersen 是内容体系的标杆** — 四层内容金字塔覆盖全部买家旅程，每个搜索意图都有对应页面
2. **国内品牌 SEO 落后 5-10 年** — 还在用 URL 数字 ID、Title 堆砌、无 Schema
3. **出口商 SEO 几乎为零** — 商业靠平台流量和直播，没有沉淀可搜索内容资产
4. **最大的机会窗口** — 英文独立站 + Andersen 级内容结构 + 出口商擅长的认证数据 = 目前市场空白
