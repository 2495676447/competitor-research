# 竞品网页剖析

> 铝合金门窗行业竞争对手全维度网页分析 — 国内品牌 + 国际品牌 + 中国出口商

## 🎯 项目定位

为新铝合金门窗项目提供竞争对手网页呈现方式的深度研究。不只是 FAQ 分析，而是整站的内容策略、页面结构、话术体系、转化路径的全景剖析。

## 📋 分析维度（8 维度）

| # | 维度 | 核心问题 |
|---|------|---------|
| 1 | 首页 | 标语、价值主张、视觉风格、CTA 策略 |
| 2 | 产品页 | 分类逻辑、规格呈现、技术参数密度、价格信号 |
| 3 | 品牌故事 | 创始人叙事、品牌支柱、认证展示 |
| 4 | 案例/项目 | 案例格式、数据展示、客户证言 |
| 5 | FAQ | 问答结构、数据密度、Schema 标记、写法特点 |
| 6 | 内容策略 | 博客主题、更新频率、B2B vs B2C 双轨 |
| 7 | 转化路径 | 采购旅程设计、表单设计、CTA 链 |
| 8 | 技术栈 | 建站平台、CDN、互动工具、性能 |
